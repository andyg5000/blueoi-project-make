DESCRIPTION:

This module provides custom functions for Medical ID Fashions.


REQUIREMENTS:
-------------



INSTALLATION:
-------------
1. Place the entire medid_custom directory into your Drupal sites/all/modules/
   directory.

2. Enable the medid_custom module by navigating to:

     administer > modules


Features:
---------
  * add price formatter display to add to cart form
